# ExtraGear 更多装备

ExtraGear 是现存的最早一批 Slimefun 附属插件之一。  
该附属向 Slimefun 添加了由各种由 Slimefun 的金属锭/合金制成的武器与防具。
这些武器与防具都带有一定的附魔。
